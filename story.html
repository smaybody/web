Prince Theodore's Journey, a choose your own adventure game

Samira M, Emily Y-S, and Shayan M, period 9

  • Project Overview

    "This program allows you to play as Prince Theodore, the heir to the kingdom of Grandworth following your expulsion from the castle by the king. You will go on a text-based journey across the kingdom, trying to make ends meet and return to the castle safely.

  • Instructions

    After running the python file, you will read through the text given to you. Then, type in or copy and paste the option you want to choose and continue accordingly. If you die along the way then feel free to start again, since the story has multiple paths and endings.

  • How the project works

    Starting the program will have it print the prologue to the story, which serves as a method to provide context for the events to come rather than being interactable with. Then, the counter function runs through a python file containing all of the possible story elements, and is used to create dictionaries detailing both the events of the story and their corresponding categorizations as well as the points that individual actions will reward the player with. From there, the program utilizes a series of branching if statements to continue the story as the player sees fit. As the player makes new choices, each new point they accumulate is added to the end of a list, the length of which is used to display their total score following the story's conclusion. Finally, in addition to a statement detailing the player's final score, the program also generates a graph based on whether they earned points or not for each choice they made during their campaign.

  • Files used

    • Final Project.py: The program the player runs in order to go through the story.

    • storylines.py: A list of all events that can take place during the course of the story; it is read into and referenced by the final project file.

    • story.html: The page you are reading now, detailing different aspects of the project

    • storystyle.css: CSS file used to style this page

  • Resources used

    The only external resource used was matplotlib, which we used to generate our graph at the end of the story
